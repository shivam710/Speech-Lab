# Speech-Lab

## Development Platforms
- Django (2.1.5)
- DBMS: Firebase

## Main Modules
- Homepage
    - Login
    - View projects 
    - View people 
    - FAQs
    - Public Notice Board
- User Management
    - User profile (Update/ Edit)
    - Personal Homepage editing (Project Addition, CV, About etc)
    - Course Viewing + enrollment
- Course Management
    - Course Profile
    - Timetable
    - Attendance Viewing + Uploading
    - Student List
    - Submission Portal
    - Assignment grading (Submission + Viewing)
- Notice Board
    - Public notices
    - Course wise notices
- Discussion Room/Forum
    - Messages + Replies
    - Polling
    - Doubt Asking / Q & A
- Calendar
    - Event Ordering
